### local 환경 사용(.env.local)
http://localhost:3000 접속
~~~shell
npm run local
~~~

<br/>

### development 환경(.env.development)
https://onyummydev.vercel.app
~~~shell
npm run build:dev
~~~
~~~shell
npm run start:dev
~~~

<br/>

### production 환경(.env.production)
https://onyummy.vercel.app
~~~shell
npm run build:prod
~~~
~~~shell
npm run start:prod
~~~

서버별 값을 세팅하고 싶으면 .env.* 파일 사용
